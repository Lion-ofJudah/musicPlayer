PlayMusic is a music player which can play audio files locally. Hopefully in the future, there will be a website just like spotify which will be integrated with the local music player. But for now I hope you enjoy it.
Currently, the application only supports:

  • .mod
  • .ogg
  • .xm
  • .mod
  • .mp3 (to a smaller extent)

As this is the first version of the application, in the next update, it might be able to support multiple music file extensions :). The seek, previous, next, favorites play and playlist play functionalities are also not fully implemented.
